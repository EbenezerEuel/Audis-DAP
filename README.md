# Audis-DAP
Combining my love for music and electronics, to create something beautiful while pursuing more knowledge.

# Introduction
The Latin word *"Audis"* roughly translates to *"you listen,"* which summarizes the goals for this project pretty well. I want to create something not only for myself but for you... to listen *see what I did there*. This is a passion project, so development will be pretty slow, but as I learn and grow, it will improve exponentially. 

# What is Audis?
*Audis* is an **100% open-source HiFi Digital Audio Player (DAP)**, focused on combining **high-end components**, **high-quality design**, and **creative design** to create something amazing.

# Features
*Audis* is "audio fidelity" first, meaning above all, squeezing the highest audio quality out of the PCB and enclosure design trumps all. Which means I will not go out of my way to implement features that take away from *Audis's* priorities, but I will try my best to make room for quality of life features that make this as consumer-friendly as possible. 

*Audis* will be based on the **STM32H7R7Z8** and the audio will be driven by the **AKM AK4452 DAC + OPA1622 amp**. *Audis* will run on custom firmware, written specifically for the MCU. Eventually, once the current concept has matured, I plan to redesign *Audis* with an ARM-based processor running a custom Linux-based OS, but this will be in the distant future. *Audis* will also integrate internal memory to provide a boost of processing power. *Audis* will have various peripherals such as **USB, MicroSD, 3.5mm Jack, various physical buttons, and a ~2.4 Inch display**. *Audis's* body will be made of **anodized steel**, but is subject to change. The goal is to have a premium-looking and feeling body, while maintaining the fidelity of the audio.
